MATH is an AngularJS app that uses a summation formula to divide 100 by any number between 0 and 200 without actually dividing at all
The yellow shows the percent accurate
The green section is the input for the calculator What number should you divide 100 by Iteration: How many times do you want to run the summation
Calculated shows the total that has been summed Actual shows the actual value
Output is the total through each iteration Added shows how much is added each iteration
